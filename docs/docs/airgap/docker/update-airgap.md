## バージョンアップ毎に新規作成する方法

### 新しいバージョンをダウンロードする

### ダウンロードしたファイルを解凍する

### 解凍したフォルダ名を変更する

### 古いエアギャップから鍵をエクスポートする

### エクスポートした鍵ファイルをコピーする

### 新しいエアギャップで鍵をインポートする

### 古いエアギャップを削除する(オプション)

## バージョンアップ時に上書きする方法

### 鍵をエクスポートしバックアップする

### 新しいバージョンをダウンロードする

### ダウンロードしたファイルを解凍する

### 解凍したフォルダを上書きする

### アップデートスクリプトを実行する
